# Machine Learning Algorithms From Scratch

Implementation of core machine learning algorithms from scratch using NumPy, with comparisons to scikit-learn.

## Algorithms
- Linear Regression (Normal Equation)
- Logistic Regression (Gradient Descent)
- Principal Component Analysis (Eigen Decomposition)

## Experiments
The models are evaluated on built-in datasets from scikit-learn:
- Breast Cancer (classification)
- Diabetes (regression)
- Iris (PCA)
